模拟登陆调试

本章节主要讲解如何模拟登陆方便调试

1 1、配置模拟登陆开关

修改配置 config/ebank.conf

%%是否启用模拟session
{login,true}. % true|false|con  代码模拟登陆|报文模拟登陆|完整流程

3 3、配置公共参数

通过修改配置config/public_params.conf 文件来配置invoke_procedure请求的公共参数，如下：

{"channel", "cs"}.
{"httpHostInfo",""}.
{"certType",""}.
{"certNo",""}.
{"mobileNo",""}.
{"platform",""}.
{"EMP_SID",""}.

注意：此处配置公共参数为invoke_procedure请求时自动补充的值，按照优先级别分别从session中的userinfo区域以及session中的transaction中的default_params中取值，每个键值对的value值表示默认值，所以为保证公共参数有值，后台开发人员仍需编写代码在合适的流程中在session中指定的区域中存放需要取出的值。
