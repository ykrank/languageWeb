2 实现思路

产品提供的柱状图扩展控件报文格式为：

<div type="plot" class='plot2' selectedType='1'>
    <ul type='yscale' class='2scale1'>
        <li>50000</li>
        ...
        <li>0</li>
    </ul>
    <ul type='xscale' class='2scale2'>
        <li>0</li>
        ...
        <li>4</li>
    </ul>
    <ul type='barPlot'>
        <li class='blueli' value='40000' onclick='ChangeInfo()'></li>
        ...
        <li class='blueli' value='50000' onclick='ChangeInfo()'></li>
    </ul>
</div>

其中xscale表示x轴，yscale表示y轴，barPlot表示非立体柱状图控件，还可以为columnPlot表示为圆形立体柱状图控件，cubePlot表示为方形立体柱状图控件，Y轴刻度值按从上到下排序，margin-top表示刻度值间距，X轴刻度值按从左到右排序，margin-top表示刻度值间距。每个柱子落在相应的x轴的刻度点上，如果柱子的个数多于x轴刻度点，则柱子按等分落在相应点上(如20个柱子，10个刻度点，每个刻度点上落两个柱子)。 filter:progid，来修改柱状图的渐变颜色。

结合需求，确定我们可以使用type为columnPlot的柱状图，使用filter:progid来作为柱体的样式，在这里先实现一个简单的柱状图。
