1 效果图

原型上效果图

从上图可以看出标题（手机银行）和两个按钮的中轴线必须是一条线，表示他们都应该是垂直居中显示在父控件中，而且标题(手机银行)应该是左右居中显示在父控件中。

根据客户端提供的规范：

  • label支持样式中的text-align可以实现标题左右居中。
  • 根据table中td的valign:middle可以实现td中控件为垂直居中。

2 正确写法

利用table中td的valign属性让客户端自动实现控件的垂直居中，以及label的text-align样式实现标题左右居中。

<content>
    <head>
        <style>
            .body{width:320px;height:460px;}
            .table_title{left:0px;top:0px;width:320px;position:fixed;}
            .tr_title{width:320px;height:51px;background-image:url(title_bg.png);}
            .label_title{width:320px;text-align:center;font-size:19px;font-weight:bold;color:#FFFFFF;}
            .button_main{left:8px;width:54px;height:32px;background-image:url(login/images/msb_scibe.png);}
            .button_back{right:8px;width:54px;height:32px;background-image:url(unlogin.png);}
        </style>
    </head>
    <body class="body" >
        <table class="table_title" name="table_title" border="0">
            <tr class="tr_title" >
                  <td align="center" valign="middle">
                     <label class="label_title">手机银行</label>
                     <input type="button" class="button_main" name="main_but" onclick="main_page_callback()"/>
                     <input type="button" class="button_back" name="back_but" onclick="back_fun()"/>
                </td>
            </tr>
        </table>
    </body>
</content>

3 错误写法

用div为父控件，然后写button和label的top值让此三个控件垂直居中，用left样式实现label的左右居中。

<content>
    <head>
        <style>
           .body{width:320px;height:460px;}
           .title_div{left:0px;top:0px;width:320px;height:51px;background-image:url(title_bg.png);position:fixed;}
           .label_title{left:123px;top:14px;font-size:19px;font-weight:bold;color:#FFFFFF;} 
           .button_main{left:8px;top:10px;width:54px;height:32px;background-image:url(login/images/msb_scibe.png);}
           .button_back{right:8px;top:10px;width:54px;height:32px;background-image:url(unlogin.png);}
        </style>
    </head>
    <body class="body" >
    <div class="title_div" border="0">
        <label class="label_title">手机银行</label>
        <input type="button" class="button_main" name="main_but" onclick="main_page_callback()"/>
        <input type="button" class="button_back" name="back_but" onclick="back_fun()"/>
    </div>
    </body>
</content>

4 注意事项

在碰到需要实现同一行多个控件中轴线平齐的时候要使用td的valign属性实现，如果使用top值来实现垂直居中会出现不同的平台和不同的分辨率显示不为居中。
