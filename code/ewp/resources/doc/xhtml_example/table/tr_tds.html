1 效果图

如图所示，本节讲述如何实现在tr中包含多个td，并利用table来实现类似图中效果的布局。

2 实现方法

示例的完整代码链接为：tr_tds.xml

项目中利用table布局的情境很多，通过把控件放在表格的tr和td中，并设置tr和td的样式，可以准确轻松的完成效果图所示的类似布局，免去定义坐标位置的麻烦。上图中所展示的效果是由上下两个table实现的，由于第二个table比较简单，这里我们只讲述第一个table实现的主要思路。

首先，我们可以看到table中有四行，即4个tr，table中包含label，input输入框，image，switch等控件，每个控件被放入一个td中，故每个tr中要包含多个td。以第一个tr为例，它是由四个td组成的，第一个td中包含label，第二个td包含input输入框，第三个td包含div(用来画竖的分隔线)，第四个td包含image。

其次，通过设置td的width值控制每个控件在行中的大致位置，通过设置控件自身的宽度和td的对齐方式还可以调整控件在td中的位置。例：我们想要switch控件距离右边框10px，而switch控件宽度为50px，则我们可以设置switch所在的td宽度为70px，并设置其对齐方式为居中align=center。

最后，画出table中的分隔线，关于分隔线的实现方法有多种，详细请参考Table分隔线一节，这里我们使用div画线的方式。横向分隔线是用height=1px的div画出的，被放在一个空的tr中。纵向分隔线是width=2px的div画出的，被放在一个空的td中。

4 注意事项

  1. 由于客户端不支持table的行列合并，故同一个表格中每个tr的td数量要相同，要用空的td补全。
  2. td中可以直接包含label，image，input，select等控件，不用嵌套div再包含这些控件。
