界面滑出动画

  • 原型效果图
  • 实现方法
  • 示例效果图

1 原型效果图

标准银行主菜单页面

点击右上方未登录按钮页面左滑，展现登录页面

如图所示为标准银行的主菜单界面，当点击右上方的未登录按钮时，界面会整体左滑，主菜单界面隐藏，显示登录界面。在登录界面用手势右滑，界面整体右滑，登录界面隐藏，回到主菜单界面。本节将讲述如何实现这样的界面滑动动画，为简便示例只实现了相似的动画，界面与标准银行不完全相同。

2 实现方法

示例的完整代码链接为：page_scroll.xml

界面滑出动画使用了transition动画接口中的translateX()函数，该函数实现了控件的x轴位移。语法如下：

transition:translateX(object,x,duration)

其中: object为要操作的控件对象；x为x轴偏移量，正值表示向右滑动，负值表示向左滑动；duration为动画持续的时间 (单位：秒)。

关于transition动画接口的详细说明请参照info_center文档中的动画。

下面我们具体分析界面滑出动画的实现方法：

与以往不同的是，我们把主界面和登录界面写在同一个页面中，分别用两个width=320px,height=460px的div实现，主界面div(name="main_page")的left坐标为0，登录界面div(name="login_page")的left坐标设为320px，故当进入界面时，只能看到主界面。页面报文代码如下：

我们给主界面中的[未登录]按钮设置了onclick事件，响应函数为translate_X()函数，该函数实现了界面向左滑动距离320px的动画效果，使得主界面隐藏登录界面显现。translate_X()的实现代码如下：

--获取主页面div控件
local main_page_ctrl = document:getElementsByName("main_page");
--获取登录页面div控件
local login_page_ctrl = document:getElementsByName("login_page");

--点击登录按钮页面沿X轴左移
function translate_X()
    transition:translateX(main_page_ctrl[1],-320,0.6);
    transition:translateX(login_page_ctrl[1],-320,0.6);
    --登录页面设置滑动监听
    gesture:setSwipeListener(login_page_ctrl[1], toMainPage);
end;

从translate_X()函数的代码中可以看到，它调用transition:translateX()接口使主界面div控件(main_page_ctrl[1])和登录界面div控件(login_page_ctrl[1])的x轴坐标都向左偏移了320px，动画持续时间为0.6s，同时它给登录页面div设置了滑动监听。

设置滑动监听的接口为gesture:setSwipeListener()，它的用法如下：

gesture:setSwipeListener(control, listener)

其中control为设置监听的控件，listener为监听方法。监听方法在定义时必须有一个参数，类型为table，table中有元素"direction" (有四个值："left"，"right"，"up"，"down")，表示滑动方向。

toMainPage()函数为这里的listener监听方法，它有一个参数argument，用户在登录界面滑动时，滑动方向的值便会传给toMainPage()函数，被存到argument["direction"]中。当 toMainPage() 判断argument["direction"]的值为"right"时，即用户在向右滑动，则调用transition:translateX()使界面向右滑动320px。这样，我们就又回到了初始的主界面。toMainPage()函数的实现代码如下：

--登录页面手势滑动，页面右移
function toMainPage(argument)
    if argument["direction"] == "right" then
        transition:translateX(login_page_ctrl[1],320,0.6);
        transition:translateX(main_page_ctrl[1],320,0.6);
    end;
end;
