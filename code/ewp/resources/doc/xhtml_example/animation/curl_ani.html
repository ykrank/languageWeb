2 实现方法

示例的完整代码链接为：curl_ani.xml

卷角动画的实现使用了timer接口，通过建立一个计时器，循环播放每一帧的图片，来达到动画的效果。

timer:startTimer()函数用来生成并启动一个计时器，语法如下：

timer:startTimer(interval, repeats, run, delay);

其参数说明如下：

  • interval(required): Number类型，表示时间间隔(单位：秒)，非法值按0处理。
  • repeats(required): Int类型，表示是否重复，1代表一直重复，0代表不重复，非法值按0处理。
  • run(required): Function类型，表示计时器循环调用方法。
  • delay(optional): Number类型，表示延迟启动的时间(秒)，如果没有指定此参数，则立即执行，非法值按0处理。

该函数会返回一个计时器对象，timer:stopTimer(timerObj)用来停止计时器使动画终止。

在实现代码中，img_more()函数响应"img_more"按钮的点击事件，生成并启动了一个计时器img_more_timer，代码如下：

--卷角动画timer对象    
local img_more_timer;

--[[按钮点击事件，开始卷角动画]]--
function img_more()
    img_more_timer = timer:startTimer(0.2,true,img_more_timer_clickFun);                             
end;

img_more_timer计时器会循环调用img_more_timer_clickFun()函数，间隔时间0.2秒。

img_more_timer_clickFun()函数实现了卷角动画，它引用了变量img_more_index来控制具体的流程。img_more_index变量相当于动画每一帧的索引值，从img_more_index=1到img_more_index=3，每个值下对应了一张图片。计时器每循环一次，便调用一次img_more_timer_clickFun()函数，将索引值加1，同时将现有图片替换为索引值对应的图片，这样就实现了卷角动画的翻页效果。当img_more_index=3时，动画播放结束，使用timer:stopTimer()停止动画，并需要将img_more_index变量重新置为0。代码片段如下：

--[[
@doc:根据名称改变控件样式
@params:
name:控件名称
style:样式名称，比如："background-image"
value:修改值，需要将控件样式修改的值
@attention:如果界面中有多个此名称控件则这些控件都会被修改。
]]--
local function changeStyle(name,style,value)
    local ctrl = document:getElementsByName(name);
    if ctrl and #ctrl > 0 then
        for key,ctrl_atom in pairs(ctrl) do
            ctrl_atom:setStyleByName(style,value);
        end;
    else
        window:alert(name .. "控件不存在！");
    end;
end;

--index
local img_more_index = 0;

--动画回调
local function img_more_timer_clickFun()

    img_more_index = img_more_index + 1;

    if  img_more_index==1 then
        changeStyle("img_more","background-image","img2.png");
    elseif img_more_index==2 then
        changeStyle("img_more","background-image","img3.png");
    elseif img_more_index==3 then
        changeStyle("img_more","background-image","img4.png");
        timer:stopTimer(img_more_timer);
        img_more_index=0;
    end;  
end;

根据以上示例总结使用timer实现动画的要点如下：

  1. 使用timer:startTimer()函数生成并启动一个计时器。
  2. 在计时器循环调用的函数中，实现动画效果。一般使用一个index变量，控制动画的步骤。注意index变量需定义在函数外。
  3. 在动画结束时使用timer:stopTimer()函数停止计时器，并重置index的值。
