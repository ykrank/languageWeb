弹出框动态布局

  • 效果图

  • 总结

1 效果图

初始状态：

点击按钮后：

如图所示，点击按钮后显示弹出框，我们假设弹出框中的文字内容是不确定的，但要求这些文字和按钮在弹出框内须垂直居中显示，本节将介绍如何实现这样的动态布局。

2 实现方法

示例的完整代码链接为：window_layout.xml

我们假定弹出框中的文字内容是不确定的，所以使用setInnerHTML动态设置了弹出框的报文。变量"text"即为弹出框label控件中的文字，可以随意改变"text"的文字看下弹出框的内容布局是否垂直居中。

页面脚本的代码片段如下：

<script type="text/x-lua">
  <![CDATA[
  --页面标记
  local tag = 1;
  --弹出框中的文字
  local text = "这是测试文字这是测试文字这是测试文字这是测试文字这是测试文字这是测试文字";

  --关闭弹出框
  function hidetable()       
    window:hide(tag);
  end;            

  --显示弹出框
  function showContent()
    --弹出框报文
    local showboby=[[
    <table class="table_showContent" name="showContent_table" border="0">
      <tr class="tr_content">
        <td class="td_content" align="center" valign="middle">
          <div class="div_showContent" border="0">
            <label class="label_info">]]..text..[[</label>
            <div class="height_10" border="0"></div>
            <input type="button" class="button_close" value="关闭" onclick="hidetable()"/>
          </div>
        </td>
      </tr>
    </table>]];

    --动态设置报文
    local table_control_show = document:getElementsByName("showContent_table");
    table_control_show[1]:setInnerHTML(showboby);
    --显示弹出框 
    window:showControl(table_control_show[1],tag); 
    --设置物理键监听
    window:setPhysicalkeyListener("backspace",hidetable);
  end;     
  ]]>
</script>

showContent()函数拼接了弹出框的报文。从弹出框的报文可以看出，它是由一个一行一列的table来实现的，在table中嵌套了一个div用来放子控件，我们利用td的align和valign属性使div水平垂直居中。由于div中的label控件的文字内容是不确定的，所以label控件和div都不能设置height值，要使它们能够随文字内容的多少自动变大变小，同时，div中子控件的top值也不能写死，要由客户端自动排版。这样，div就能根据label中文字的内容适应性变化大小，又因为td设置了valign="middle"属性，所以div整体会垂直居中显示。

弹出框的CSS样式代码片段如下：

<style>
    ...
    .table_showContent{width:280px;height:300px;top:100px;left:20px;background-color:#FFFFFF;display:none;}
    .tr_content{height:300px;}
    .td_content{width:280px;}
    .div_showContent{width:220px;background-color:#FFFFFF;}  
    .label_info{width:220px;font-size:18px;font-weight:bold;}
    .button_close{left:10px;width:200px;height:40px;background-color:#0068b7;font-size:18px;color:#FFFFFF;}
    .height_10{height:10px;}
</style>

代码中可以看到：div_showContent和label_info未设置height值，高度动态变化；div及其子控件都未设置top值，由客户端自动排版。

3 总结

当弹出框中内容不定时，要实现其内容垂直居中的思路如下：

  1. 用table嵌套div实现弹出框，利用table的valign属性使div垂直居中；
  2. div不能设置top值和height值，须根据内容动态变化；
  3. div中的子控件不能写top值，使用客户端自动排版。
