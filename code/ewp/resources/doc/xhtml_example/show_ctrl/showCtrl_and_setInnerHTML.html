showControl和setInnerHTML同时使用

1 效果图

如图所示为动态生成的弹出框，是由setInnerHTML(动态设置报文)和showControl(显示弹出框)共同实现的。本节将介绍showControl和setInnerHTML同时使用的情况。

2 实现方法

示例的完整代码链接为：showCtrl_and_setInnerHTML.xml

2.1 代码思路

示例中的弹出框是由setInnerHTML和showControl接口共同实现的：setInnerHTML接口用来设置弹出框的报文；showControl接口使该弹出框显示。showControl要跟hide配合一起使用，一个tag对应一个悬浮的页面，当showControl使用一个tag打开页面后，必须使用hide关闭该tag的页面。

示例的界面报文如下：

<body>
    <!--主页面-->
    <label class="label_title">showControl和setInnerHTML同时使用</label>   
    <input type="button"  class="button_showContent" value="显示弹出框" onclick="contentShow()"/>

    <!--弹出框-->
    <div class="div_showContent" name="showcontent_div" border="0"></div>                    
</body>

name为"showcontent_div"的div即为弹出框，它开始是空的且被隐藏。当点击界面上的"显示弹出框"按钮后，会触发contentShow()函数，该函数调用setInnerHTML方法给"showcontent_div"动态设置了报文，并调用showControl方法使其悬浮在主页面上。

contentShow()函数的代码如下：

--页面标记
local tag = 1;

--显示弹出框            
function contentShow()
  --弹出框报文
  local showboby=[[
    <div class="div_showContent" name="showcontent_div" border="0">
        <label class="label_info">这是被隐藏的div！</label>
        <input type="button" class="button_close" value="关闭" onclick="hidetable()"/>
    </div>
  ]]
  --动态设置报文
  local div_control_show = document:getElementsByName("showcontent_div");
  div_control_show[1]:setInnerHTML(showboby);
  --显示弹出框 
  window:showControl(div_control_show[1],tag); 
  --设置物理键监听
  window:setPhysicalkeyListener("backspace",hidetable);       
end;

"showcontent_div"报文更新后，弹出框中包含一个"关闭"按钮，点击该按钮会触发hidetable()函数使该弹出框隐藏。

hidetable()函数的代码如下：

--关闭弹出框
function hidetable()       
  window:hide(tag);
end;

可以看到，hidetable()调用window:hide()隐藏了标记为tag的页面，该tag与showControl中的tag相对应。

2.2 错误写法

有如下错误的报文写法：

--显示弹出框            
function contentShow()
  ...
  --获取控件
  local div_control_show = document:getElementsByName("showcontent_div"); 
  --showControl
  window:showControl(div_control_show[1],tag);
  --setHTML
  div_control_show[1]:setInnerHTML(showboby);
  --改变"display"样式
  changeStyle("showcontent_div","display","block");
  ...       
end;

--改变控件样式
function changeStyle(name,style,newValue)
  local objs = document:getElementsByName(name);
  if objs and #objs > 0 then
      objs[1]:setStyleByName(style,newValue);
  end;
end;

以上报文有两点错误：

  1. setInnerHTML在showControl后使用，这样可能会导致错误或者界面闪动。
  2. 在showControl后又使用了改变样式的方法，showControl本身就使隐藏的div显示了，不需要再改变其"display"属性。在showControl后再使用改变样式的方法，可能会刷新界面导致屏幕闪动。

3 注意事项

  1. 当showControl和setInnerHTML一起使用时，必须先使用setInnerHTML设置报文，再使用showControl。
  2. showControl后尽量不要调用改变样式的方法。
