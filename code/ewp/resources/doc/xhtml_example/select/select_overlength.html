select控件长度超长

  • 效果图
  • 实现方法
    • 代码思路
    • 错误示范
  • 注意事项

1 效果图

如图所示，当select控件中的文字内容超出长度时，客户端会将超出部分显示为"..."。本节将实现图中的界面布局并指出select控件长度超长时需注意的问题。

2 实现方法

2.1 代码思路

示例的完整代码链接为：select_overlength.xml

示例的界面布局是由table实现的，table中总共有6个tr，其中三个tr是为了实现分隔线（分隔线是用高度为1px的div画的），其他三个tr即为我们看到的表格中的三行。每行只有一个td，td中有多个子控件，通过设置宽度来控制它们在行中的位置。在第一行中，包括一个label控件和一个select控件，为了使两个控件显示在同一行，须分别给它们设置width宽度，且两个控件的width值加起来要小于td的宽度。注意：td中的子控件与td间有间距，计算宽度时要考虑这个距离。如若未设置select控件的width值，当选项的文字特别多时，select控件会适应变长导致换行显示或超出自己应占的区域。为了使label控件和select控件垂直居中，两个控件最好都设置相同的height值。手机号和查询密码的布局方式与账号相同。

CSS代码片段如下：

<style> 
    .body{background-color:#FFFFFF;}
    .table1{width:300px;top:30px;left:10px;}
    .tr1{height:45px;}
    .td{width:300px;}
    .label{font-size:15px;height:30px;}
    .label1{width:60px;}
    .label2{width:60px;}
    .label3{width:80px;}
    .select_accountNo{right:10px;font-size:15px;color:#000000;width:230px;height:30px;text-align:right;background-color:#F0F0F0;}
    .input{right:10px;height:30px;text-align:right;}
    .input1{width:220px;}
    .input2{width:200px;}
    .line{height:1px;background-color:#F0F0F0;}
</style>

页面报文代码如下：

<body> 
  <table class="table1" border="0"> 
    <tr class="tr1"> 
      <td>
        <label class="label1">账号：</label> 
        <select class="select_accountNo"> 
          <option>账户1234569311123483722222842</option> 
          <option>3703031231231239999</option> 
        </select>
      </td>
    </tr>
    <tr>
      <td><div class="line"></div></td>
    </tr>
    <tr class="tr1"> 
      <td>
        <label class="label2">手机号：</label> 
        <input type="text" class="input1" hold="请输入" border="0" style="-wap-input-format:'phone'"/>
      </td>
    </tr>
    <tr>
      <td><div class="line"></div></td>
    </tr>
    <tr class="tr1"> 
      <td>
        <label class="label3">查询密码：</label> 
        <input type="text" class="input2" hold="请输入" border="0"/>
      </td>
    </tr>
    <tr>
      <td><div class="line"></div></td>
    </tr>
  </table>
</body>

2.2 错误示范

  1. select未设置width和right值，被换行显示。
    CSS如下：
     .select_accountNo{font-size:15px;color:#000000;height:30px;text-align:right;background-color:#F0F0F0;}
    效果图如下：
  2. select未设置width值有right值，label被select遮盖。
    CSS如下：
     .select_accountNo{right:10px;font-size:15px;color:#000000;height:30px;text-align:right;background-color:#F0F0F0;}
    效果图如下：
  3. select设置的width值与label的width值之和正好等于td的宽度，select控件换行显示。
    CSS如下：
     .label1{width:60px;}
 .select_accountNo{font-size:15px;color:#000000;width:240px;height:30px;text-align:right;background-color:#F0F0F0;}
    效果图如下：
  4. select和label都未设置height值。
    CSS如下：
     .label{font-size:15px;}
 .select_accountNo{font-size:15px;color:#000000;text-align:right;background-color:#F0F0F0;}
    效果图如下：

3 注意事项

在使用select控件时，要考虑到选项文字特别多的情况，在这种情况下，若select控件未给定宽高，控件会适应性变大。我们需要根据情况决定是否需要给select控件设置width和height值，以得到我们想要的布局。当select控件的宽度不能完全显示选项文字时，会将超出的文字部分显示为"..."。
