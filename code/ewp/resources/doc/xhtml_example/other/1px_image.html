图片高度为1px

  • 问题描述
  • 解决方法

1 问题描述

当img控件的src属性所给的图片高度为1px时，在某些型号的手机上显示不出来。

2 解决方法

使用高度为2px的图片，在样式中将img控件的height设为1px，图片即能正常显示。

以下为测试的报文： 

<?xml version="1.0" encoding="UTF-8" ?>
<content>
  <head>
    <style>
    .body{background-color:#F0F0F0;}
    .label1{left:0px;top:40px;width:320px;height:40px;font-size:18px;font-weight:bold;text-align:center;}
    .img_dotted_line{height:1px;width:260px;left:20px;}
    </style>
  </head>
  <body>
    <label class="label1">img图片高度为1px</label>
    <img src="local:dotted_line.png" class="img_dotted_line"/>
  </body>
</content>
