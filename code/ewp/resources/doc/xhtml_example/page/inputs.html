1 效果图

由多个输入框组合构成界面，每个输入框只有hold提示，前面没有label文本。

2 实现方法

2.1 简洁写法

直接写几个input控件自动从上往下排版即可。

实现代码：

CSS:

    .first_input{left:8px;top:30px;width:304px;height:40px;font-size:15px;background-color:#FFFFFF;}
    .second_input{left:8px;top:77px;width:304px;height:40px;font-size:15px;background-color: #FFFFFF;}
    .third_input{left:8px;top:124px;width:304px;height:40px;font-size:15px;background-color: #FFFFFF;}

XHTML:

<!--转入账户-->    
<div class="accInfo_div" name="div_accInfo" border="0">
    <label class="label_tip">转入帐户</label><br/>
    <div class="div_payee" name="payee_table" border="0" onclick="request_page('fast_transfer','mb0103')">
        <label class="label_payee" name="label_payee">我的常用收款人</label>
    </div>
    <div class="first_table" border="0" onclick="request_page('fast_transfer','mb0102')">
        <label class="label_bankname" name="bank_name">请选择收款银行</label>
        <img src="local:fast_transfer/images/card_pull_but.png" class="image_arrow"/>
    </div>
    <input type="text" class="second_input" value="" name="name" border="0" hold="请输入收款人姓名" maxleng="20"/>
    <input type="text" class="third_input" value="" name="accNo" border="0"  style="-wap-input-format:'N'" hold="请输入收款人帐号" maxleng="20"/>
</div>

<!--其他转账信息-->
<div class="info_div" name="div_info" border="0">
    <label class="label_tip">其他转帐信息</label>
    <input type="text" class="first_input" value="" name="amount" style="-wap-input-format:'n'" maxleng="9" border="0" hold="请输入转帐金额"/>        
    <input type="text" class="second_input" value="" name="uses" border="0" hold="请输入转帐用途" maxleng="20"/>
    <input type="text" class="third_input" value="" name="mobileNo" style="-wap-input-format:'phone'" maxleng="11" minleng="11" border="0" hold="请输入收款人手机号"/>        
</div>

2.2 复杂写法

把输入框包含在table中，每一个tr中包含一个input。

实现代码：

      <label class="label_tip">转入帐户</label>
      <table class="table_payee" name="payee_table" border="0">
        <tr class="tr_payee" onclick="request_payee()">
              <td align="center" valign="middle">
                <label class="label_payee" name="label_payee">我的常用收款人</label>
              </td>
        </tr>
      </table>
      <table class="first_table" border="0">
