1 介绍

引入CS模板系统之后，EWP后台所有与展现逻辑相关的业务流程，其输入可分为三个部分 ：1、CS模板文件；2、业务请求所返回的XML数据；3、部分业务相关的参数，其中2和3为可选（除纯静态页面外，两者必有其一）。

在系统初始化时，我们会将所有的模板文件读入，并以文件名为索引，保存到driver中的hash map中。我们提供更新的接口，以动态改变模板文件的内容。

当业务进行到展示逻辑时，可调用CS driver的render接口，将CS文件名（作为hash map的索引值）和相关参数（包括远程返回XML和业务参数等）发送到driver，经过CS库调用后返回生成 rendered binary，再由服务器发送给客户端。

CS Driver提供两个重载函数来生成页面数据。cs_api:render/2和cs_api:render/3。

以下对其进行详述：

2 API Functions

2.1 1.render(CsKey, Input)

描述：根据指定模板及输入的tuplelist生成字符串。指定模板只须通过文件名前缀，如通过"tmpl"指定"app_dir/public/cs/tmpl.cs"文件。参数Input是一个tuplelist，描述了模版所需要的参数。 例：cs_api:render("tmpl", [{"key1", "value1"}])，以及在模板中可通过#{cs var:key1}#获取value1。

render(CsKey, Input) ->
    ?LOG(tmpl:parsed_render(Input, CsKey)).

参考类型：

(CsKey :: string(), Input :: input()) -> binary()
input() = [{key(), value()}]
key() = string() | atom()
value() = string() | atom() | integer() | input()

2.2 2.render(CsKey, Xml,Input)

描述：根据指定模板及输入的tuplelist和xml生成字符串。参数Xml是标准的xml数据，描述了模版所需要的参数。 例：cs_api:render("calendar",<<"123">>,[{"key1","value1"}]). 以及在模板中可通过#{cs var:root.value}#获取value。

render(CsKey, Xml,Input) when is_binary(Xml)->
    ?LOG(tmpl:parsed_render(Input, Xml, CsKey)).

参考类型：

(CsKey :: string(), XML :: binary(), Input :: input()) -> binary()

3 举例

(1)编辑/mnt/hgfs/work/testcs.cs文件。

示例代码：
Start test: <?cs var:hello ?>
If test:
<?cs if:flag ?>
flag is <?cs var:flag ?>
<?cs else ?>
flag is not exist
<?cs /if ?>
Each test:
<?cs each:r = records.record ?>
record: <?cs var:r ?>
<?cs /each ?>

(2)添加cs模版文件：

(ewp@tracvm)1>cs_api:add_cs_files(["/mnt/hgfs/work/testcs.cs"]).

(3)编译运行效果为：

a.编译运行render/2：
(ewp@tracvm)2>cs_api:render("testcs", [{flag, 123},{hello,"hello cs"},{records,[{record, 1}, {record, "test rec"}]}]).
效果：<<"Start test: hello cs If test:  flag is 123  Each test:  record: 1  record: test rec ">>

b.编译运行render/3：
(ewp@tracvm)3>cs_api:render("testcs", <<"<?xml version=\"1.0\" encoding=\"utf-8\"?><records><record>123</record><record>abc</record><records>">>, [{flag, 123},{hello,"hello cs"}]).
效果：<<"Start test: hello cs If test:  flag is 123  Each test:  record: 123  record: abc ">>

(4)移除cs模版文件：

(ewp@tracvm)4>cs_api:del_cs_files(["/mnt/hgfs/work/testcs.cs"]).
