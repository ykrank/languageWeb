3.0项目实施规范

开发规范

基于EMP的移动应用软件开发按照团队分工主要分为

  1. 前端开发，主要包括应用的UI展现部分，基于ERT客户端、前端框架和前端开发技术规范实现
  2. 后端开发，主要包括应用的业务逻辑处理，基于EWP和Erlang开发实现
  3. 客户端定制，主要包括基于各客户端平台原生代码开发的扩展控件，例如指纹识别、摄像头、定制UI等

一般，前端开发占据项目实施中大部分开发工作量。根据前端开发所选用的不同技术规范，又可以 分为EMP和HTML两种实现方式。

针对上述内容，我们分别定义了如下的开发手册（含规范）：

  1. 前端开发手册
  2. EWP开发手册
  3. 客户端定制控件开发手册
