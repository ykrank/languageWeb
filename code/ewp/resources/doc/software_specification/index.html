3.0项目实施规范

《3.0项目实施规范手册》描述基于EMP5.3开发手机银行类项目软件系统的整体规范。 规范从类别上可以划分为：

  1. 技术类规范，主要针对软件系统所使用到的技术架构和技术实现，如《系统安全规范》
  2. 实施类规范，主要针对实施团队的开发流程，如《前端UI复用规范》

其中技术类规范按层级又可以划分为：

  1. 系统架构类，主要针对系统安全、稳定性和扩展性
  2. 编码开发类，主要针对代码质量、可复用性和可维护性

规范手册中的内容适用于基于EMP5.3开发的移动应用，包含EMP和标准HTML两种不同的前端开发技术规范。
