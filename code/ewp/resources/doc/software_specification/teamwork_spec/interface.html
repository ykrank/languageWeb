3.0项目实施规范

接口开发规范

  1. 前后端接口定义由EWP开发人员驱动，在前端开发之前完成Channel（业务功能逻辑单元）及接口定义；
  2. 接口返回的模拟报文通过开发工具集成，使得前后端开发可以并行进行，非业务需求变更的情况下， 不允许对接口报文进行随意变更；
  3. 在前、后端分别完成业务逻辑的开发和自测后，进入前后端联调测试环节；
