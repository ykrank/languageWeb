2 兼容性问题

2.1 后台版本兼容问题

由于离线协议4.0的引入，当前版本（V5.4.24B）客户端无法兼容低于5.4版本的后台服务器。

2.2 　布局算法缩放公式修改

在5.3版本，计算页面控件布局时，根据屏幕适配缩放可能会得到小数坐标，四舍五入的话，可能会有显示问题，因此5.3统一向下取整将小数舍掉了。
5.4优化了布局算法，解决了显示问题。小数部分会四舍五入得到最终坐标。
对控件布局效果无影响，但对label文本的适配可能会造成比之前显示稍大的影响。
