Description

CS模板相关API。CS模板是EWP提供的字符串处理工具，其处理效率优于 Erlang的字符串处理。在处理动态数据生成较长的字符串时，使用CS模 板会提升后台运行效率。开发者可通过.app文件的cs_dirs配置项配置 CS模板所在根目录。
关于CS基础的可参考 ClearSilver Documentation。

Function Details

render/2

render(CsKey::string(), Input::input()) -> binary()

根据指定模板及输入的tuplelist生成字符串。 指定模板只须通过文件名前缀，如通过"tmpl"指定"app_dir/public/cs/tmpl.cs"文件。
例：cs_api:render("tmpl", [{"key1", "value1"}])，在模板中可通过#{cs var:key1}# 获取value1。

render/3

render(CsKey::string(), XML::binary(), Input::input()) -> binary()

根据指定模板及输入的tuplelist和xml生成字符串。
例：cs_api:render("tmpl", <<"<root><value>2</value></root>">>, [])， 在模板中可通过#{cs var:root.value}#获取value1。
