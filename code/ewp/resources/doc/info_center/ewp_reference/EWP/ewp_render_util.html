http响应生成通用API。该模块用来生成yaws规定的http响应格式， 调用者无须关注API的具体返回格式。该模块API常用来生成自定义 controller的返回值。.

Description

http响应生成通用API。该模块用来生成yaws规定的http响应格式， 调用者无须关注API的具体返回格式。该模块API常用来生成自定义 controller的返回值。
