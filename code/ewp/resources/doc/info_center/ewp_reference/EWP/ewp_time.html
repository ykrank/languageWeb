date_to_gsec/1

date_to_gsec(Date::string() | binary() | {datetime, datetime()}) -> integer()

把日期和时间转化成公历时间秒数. 如果参数是字符串，它有可能是以下情况之一: "2007-11-04 15:30:00 +0000"，"2007-11-04 15:30:00"， "2000-03-04T23:00:00.1234560+03:00"和"20061206T120000". 请参考 calender module获取更多时间信息.
