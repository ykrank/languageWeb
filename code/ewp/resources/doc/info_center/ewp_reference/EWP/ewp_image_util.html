download_shrink_image_with_aspect/3

download_shrink_image_with_aspect(Url::string(), Width::integer(), Height::integer()) -> binary()

throws image_download_failed | string()

根据指定url下载图片，并按指定宽高压缩图片再转换为PNG格式。
与download_shrink_image/3不同的是，该函数不会改变图片的宽高比。
如果指定宽高均比图片宽高值大的话，图片不会被压缩。否则，将对比 指定宽与图片宽比值和指定高与图片高比值，按较小比例压缩图片。
