1 前言

本文档记录5.3相对于5.2在控件和lua脚本上的变更范围，具体修改内容请点击链接进相关页面查看。

2 属性

2.1 新增属性

全局属性

  • id

其他属性

  • adjustsfontsize
  • autoplay
  • clearmode
  • cornerradius
  • delay
  • failed
  • isorder
  • istap
  • leftimg
  • leftimgmode
  • linebreakmode
  • loading
  • minfontsize
  • offtintcolor
  • ontintcolor
  • returnkeyaction
  • returnkeytype
  • rightimg
  • rightimgmode
  • shadowcolor
  • shadowoffset
  • substitute
  • thumbtintcolor
  • titleimg
  • velocity

2.2 属性变更

  • checked只支持checked、true、false三种值，其中checked和true都表示选中，false表示不选中。其余均为非法值
  • enable设置非法值时，按默认值true处理
  • 原encryptMode改为小写encryptmode
  • 原showFormat改为小写showformat，默认格式统一为yyyyMMdd
  • 原valueFormat改为小写valueformat，默认格式统一为yyyyMMdd

3 样式

3.1 新增样式

全局样式

  • position:topLevel

其他样式

  • border-radius
  • font-style
  • inline-block
  • visibility

3.2 样式变更

  • 增加伪类支持
  • text-align样式取消相对于父控件布局的功能

4 页面控件标签

4.1 新增标签

  • Richtext

4.2 标签变更

注：

  1. 新增的全局属性和样式在各控件标签中不再赘述；
  2. 控件默认宽高均以320*480为报文基础分辨率。

4.2.1 A

  • 新增支持linebreakmode属性
  • 取消支持value属性
  • 新增支持伪类
  • 默认宽高统一为根据文字内容计算，但宽度不可超出父控件显示范围

4.2.2 Br

  • 默认高度统一为35px

4.2.3 Div

  • 新增支持font-style、border-radius、inline-block样式
  • 默认宽高统一为根据子控件内容计算，但宽度不可超出父控件显示范围

4.2.4 Img

  • 新增支持gif格式图片
  • 新增支持autoplay、istap、loading、failed属性
  • 默认宽度统一为30px，默认高度统一为30px

4.2.5 InputButton

  • 新增支持delay、leftimg、rightimg属性
  • 新增支持font-style、border-radius、visibility样式
  • 新增支持伪类
  • 默认宽高统一为根据文字内容计算，但宽度不可超出父控件显示范围

4.2.6 InputCheckbox

  • 新增支持linebreakmode、numlines、shadowcolor、shadowoffset、valign属性
  • 新增支持background-color、background-image、filter、font-style样式
  • 默认宽高统一为根据文字内容计算，但宽度不可超出父控件显示范围

4.2.7 InputPassword

  • 新增支持isorder、substitute属性
  • 新增支持border-radius样式
  • 默认宽度统一为300px，默认高度统一为24px

4.2.8 InputRadio

  • 新增支持linebreakmode、numlines、shadowcolor、shadowoffset、valign属性
  • 新增支持background-color、background-image、filter、font-style样式
  • 默认宽高统一为根据文字内容计算，但宽度不可超出父控件显示范围

4.2.9 InputSegment

  • 新增支持cornerradius、ontintcolor、titleimg属性
  • 新增支持font-style、visibility样式
  • 默认宽度统一为180px，默认高度统一为30px（iOS平台除外）

4.2.10 InputSwitch

  • 新增支持offtintcolor、ontintcolor、thumbtintcolor属性
  • check属性设置非法值时，按默认值false处理
  • 默认宽度统一为120px，默认高度统一为30px（iOS平台除外）

4.2.11 InputText

  • 新增支持clearmode、leftimg、leftimgmode、rightimg、rightimgmode、returnkeyaction、returnkeytype、
  • 新增支持border-radius、font-style样式
  • 默认宽度统一为300px，默认高度统一为24px

4.2.12 Label & I & Em & B & Strong

  • 新增支持adjustsfontsize、linebreakmode、minfontsize、shadowcolor、shadowoffset、velocity属性
  • 取消支持value属性
  • 新增支持font-style样式
  • 默认宽高统一为根据文字内容计算，但宽度不可超出父控件显示范围

4.2.13 Select & Option

  • select新增支持border属性
  • select新增支持border、border-radius、font-style样式
  • option新增支持color、font-size、font-style、font-weight、text-align样式
  • select默认宽度统一为根据内容最长的子option计算，，但不可超出父控件显示范围，默认高度统一为30px

4.2.14 Table

  • 新增支持border-radius、font-style样式
  • 默认宽高统一为根据子控件内容计算，但宽度不可超出父控件显示范围

4.2.15 Td

  • 新增支持font-style样式
  • 默认宽高统一为根据子控件内容计算，但宽度不可超出父控件显示范围

4.2.16 Tr

  • 新增支持font-style样式
  • 默认宽高统一为根据子控件内容计算，但宽度不可超出父控件显示范围

6 Lua接口

6.1 新增接口

  • document:getElementById(id)
  • document:getElementsByClassName(className)
  • document:createElement(tagname, property)
  • kv:put(key, value)
  • kv:get(key)
  • kv:del(key)
  • offline:checkOfflineFileWithServer(filename)
  • control:appendChild(control1)
  • control:insertBefore(control1, control2)
  • control:removeChild(control1)

6.2 接口变更

  • location:reload()增加可选参数，改为location:reload(reset)
  • offline:update_resource()增加可选参数，改为offline:update_reource(params)
  • offline:checkOfflineFile(filename)针对可选资源的检验规则变更
  • window:showControl(object, tag, transitionType)增加可选参数，改为window:showControl(object, tag, transitionType, isModel)
  • control:getPropertyByName()接口支持获取所有属性，包括自定义属性
