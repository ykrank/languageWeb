1 Session操作函数

1.1 new_session/1

  • Description:
    创建一个session 对象和SessionID。
  • Syntax: 
    new_session(PropList::list()) -> SessionID::string()

1.2 get_session_id/1

  • Description:
    在yaws arg中获取session id
    • Syntax: 
      get_session_id(YawsArg::#arg{}) -> SessionID::string()

1.3 read/0

  • Description:
    读取当前session信息
  • Syntax: 
    read() -> {SessionID::string(), PropList::list()} | {SessionID::string(), undefined} | {undefined, undefined}

1.4 read_by_id/1

  • Description:
    根据指定sessionID读取session内容。
  • Syntax: 
    read_by_id(SessionID::string()) -> Proplists::list() | undefined

1.5 update/2

  • Description:
    更新session数据。 **
  • Syntax: 
    update(SessionID::string(), PropList::list()) -> ok

其他相关Function请参照参考手册

2 Examples

以登录为例子介绍将登录信息存入Session中，在账户查询时将登录信息从Session中取出使用。

2.1 登录

我们先将login方法放在training_controller中。

实现Function:login

%% @doc:登录
%% @params:#arg{}

login(A) ->
    P =  ewp_params:from_yaws_arg(A),
    %% 登录输入
    CardNo = ewp_params:get("cardNo",P),
    Password = ewp_params:get("password",P),
    CardFlag = ewp_params:get("cardFlag",P),

    %% 暂定为登录成功，返回为xml数据，中间的请求第三方或者查询数据库验证等过程先省略。

    ReturnSuccess = 
        "<?xml version=\"1.0\" encoding=\"utf-8\"?>
            <Message>
                <Head>
                    <ReturnCode>000000</ReturnCode>
                    <ReturnMsg></ReturnMsg>
                </Head>
                <Body>
                    <IsToken>0</IsToken><!--客户端认证方式-->
                    <IsTranfer>1</IsTranfer><!--客户是否开通转账汇款功能-->
                    <UserId>000001</UserId>
                    <UserName>张三</UserName><!--用户姓名-->（首次登陆登录下面三个信息为空）
                    <CheckInfo>我的地盘我做主</CheckInfo><!--预留信息--> 
                    <simplePwdFlag>0</simplePwdFlag>
                    <UserTodayLoginTimes>10</UserTodayLoginTimes><!--—今天登录次数--> 
                    <LastLogonDatetime>20140405</LastLogonDatetime><!--上次登录时间-->
                </Body>
            </Message>",
    %% 解析xml获取用户信息，然后存入Session中。
    ReturnTerm = xml_eng:xml_to_term(ReturnSuccess),

    [UserId] = ?ewp_xpath("Message/Body/UserId",ReturnTerm),
    [UserName] = ?ewp_xpath("Message/Body/UserName",ReturnTerm),
    ?ewp_log("UserName is =======~p~n",[UserName]),
    [CheckInfo] = ?ewp_xpath("Message/Body/CheckInfo",ReturnTerm),
    [UserTodayLoginTimes] = ?ewp_xpath("Message/Body/UserTodayLoginTimes",ReturnTerm),
    [LastLogonDatetime] = ?ewp_xpath("Message/Body/LastLogonDatetime",ReturnTerm),
    ?ewp_log("UserTodayLoginTimes,LastLogonDatetime is ======~p~p~n",[UserTodayLoginTimes,LastLogonDatetime]),
    %% 新建Session
    PropList = [{"UserId",UserId},{"UserName",UserName},{"CheckInfo",CheckInfo},
                {"UserTodayLoginTimes",UserTodayLoginTimes},
                {"LastLogonDatetime",LastLogonDatetime}],

    SessionID = session_service:new_session(PropList),
    ?ewp_log("SessionID is =========~p~n",[SessionID]),

    Cookie = session_service:make_cookie_str(SessionID),

    %% 返回登录成功json数据
    Ret_json = xml_eng:xml_to_json(ReturnSuccess),

    %% 返回浏览器SessionID，便于下次请求时获取此ID
    ewp_render_util:render_json(Ret_json,[Cookie]).

浏览器中请求： http://192.168.241.3:4002/training/login?CardNo=11111&password=0993&cardFlag=1

2.2 账户查询

修改Module开发章节时介绍的query_accountFunction中UserId的获取方式为从Session中获取。 即将

P =  ewp_params:from_yaws_arg(A),
UserId = ewp_params:get("UserId",P),

修改为

    SessionId = session_service:get_session_id(A),
    ?ewp_log("SessionId is ====~p~n",[SessionId]),

    SessionData = session_service:read_by_id(SessionId),
    %% 添加一些容错代码
    UserId = case SessionData of
                undefined ->
                    throw("Session 信息不存在。");
                _ ->
                    o(SessionData,"UserId")
            end,
    ?ewp_log("UserId is ===========~p~n",[UserId]),

在浏览器中请求： http://192.168.241.3:4002/training/query_account?UserId=000002

此时不管传入的UserId为多少，返回都为图片中所示账户列表。
