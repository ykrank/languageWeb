离线资源使用说明文档

1 离线资源预置说明

1.0.1 离线协议版本2.1及之前的版本

  1. 离线资源包下载到本地.
  2. 将资源包中的zip包文件全部解压，解压后删除zip包.
  3. 拷贝资源包中的资源，粘贴到工程中的offline文件夹下.

1.0.2 离线协议版本3及之后的版本

  1. 分别将需要的多个离线资源包下载到本地.
  2. 将资源包中的zip包文件全部解压，解压后删除zip包.
  3. 找到工程中的offline文件夹目录
  4. 根据要预置的文件的appName的不同，分别在offline文件下，创建名字为相应appName的文件夹.
  5. 拷贝对应离线资源包中的文件，将之粘贴到与之对应的新创建的appName文件夹下。

2 离线开发部分方法使用说明

2.0.1 离线3.0及之后的版本部分方法使用说明(与之前版本的不同说明)

开发常用方法不同一览表

版本号 < 3 版本号 >= 3 方法说明
-(void)updateDesc; - (void)updateDesc:(NSString *)appName; desc描述请求接口
-(void)updateResource; -(void)updateResource:(NSString *)appName; 离线资源请求下载接口
-(void)downloadSingleResourceWithPath:(NSString *)downloadPath; -(void)downloadSingleResourceWithPath:(NSString )downloadPath andAppName:(NSString )appName; 发起离线请求，返回请求数据
-(NSString *)getPort:(BOOL)optional; -(NSString )getPort:(BOOL)optional withAppName:(NSString )appName 获取port端口号
-(NSString *)getSecureTransType; - (ResponseServerModel )getResponseServerModelWithAppName:(NSString )appName;
-(NSString )getSecureTransTypeWithRespomseServeMode:(ResponseServerModelVerThird )responseServerModel;		 获取SecureTransType
-(NSString *)getClientHash; -(NSString )getClientHashWithAppName:(NSString )appName; 获取必选资源的hash值
-(NSString *)getOptionServerHash; -(NSString )getOptionServerHashWithAppName:(NSString )appName 获取可选资源的hash值
-(NSDictionary *)getOptInfoInServer; -(NSDictionary )getOptInfoInServer:(NSString )appName; 获得当前本地保存的服务器描述文件中所有可选文件描述信息的table
-(NSDictionary *)getOptInfoInLocal; -(NSDictionary )getOptInfoInLocal:(NSString )appName 获得当前本地保存的服务器描述文件中所有已下载可选文件描述信息的table

2.1新增接口

方法名 方法说明
- (NSData )dataFromH5FileName:(NSString )name appName:(NSString *)appName; 根据文件名和appName参数，获取本地存储h5文件的二进制数据
- (NSString )getMimeType:(NSString )name appName:(NSString *)appName 根据文件名和appName参数，获取本地存储文件的MimeType属性 
注：上面两个接口，目前传入的appName没做处理，3.0之前的版本可传nil.之后的版本采用的是全局默认的appName。
