4.1 实现类 WebSocketDownload

说明：

4.1.1 构造方法

public WebSocketDownload(FlowManagerImpl flowManager, String ws, int port) {
    super(flowManager, ws, port);
    mWS = ws;
    mTramode = mTramode == 3 ? 0 : mTramode; // 暂定如果st为3 则为明文传输

    mWebSocket = new WebSocketConnection();
    mWSOptions = new WebSocketOptions();
    // 设置最大下载文件大小(目前为5M)
    mWSOptions.setMaxFramePayloadSize(5 * 1024 * 1024);
    mWSOptions.setSocketConnectTimeout(60000);// 设置1分钟连接超时
}

说明：

  • 目前暂定如果安全传输方式为3时 则按照0 （明文传输方式）处理。
  • 设置最大下载文件大小为5M

4.1.2 连接服务器

public WebSocketDownload init() {

    try {
        mIsConnecting = true;

        mWebSocket.connect(mWS + Constants.OTA_WS, new WebSocketConnectionHandler() {

            @Override
            public void onOpen() {
                Utils.printLog(Constants.OFF_TAG, "Status: connected to...");

                mIsConnecting = false;

                if (mIsDownloading) { // 服务器器重连成功，处理待请求的Model
                    mWebSocket.sendBinaryMessage(mBodys);
                } else  {
                    notifyConnecting();
                }
            }

            @Override
            public void onBinaryMessage(byte[] bytes) {

                mBytes = handleResponse(bytes);
                mIsDownloading = false; // 下载成功

                notifyDownloading();
            }

            @Override
            public void onClose(int code, String reason) {
                Utils.printLog(Constants.OFF_TAG, "Status: closed...[" + code + " : " + reason + "]");

                mIsConnecting = false;

                if (mIsDownloading) {// 若资源下载时需要重连，若主动断开不用重连
                    reconnectServer();
                } else {
                    notifyConnecting();
                }
            }

        }, mWSOptions);
        return this;
    } catch (WebSocketException e) {
        Utils.printLog(Constants.OFF_TAG, "Status: connect exception...");
        Utils.printException(e);
    }
    return null;
}

说明：

  • public void onOpen() {}
    连接websocket 服务器成功调用该方法，向服务器发送请求需要在该方法之后
  • public void onClose(int code, String reason) {}
    websocket连接关闭时调用该方法，无论服务器关闭还是客户端主动关闭均会调用
  • public void onBinaryMessage(byte[] bytes) {}
    接收到服务器返回的内容调用该方法。目前服务器返回均是byte[] 格式所以我们使用该方法。如果返回String 类型则需使用 public void onTextMessage(String payload) {}方法

@Override

    mBytes = null;

4.1.4 叫醒等待websocket 连接的线程

/**
 * 叫醒等待连接的线程
 */
private void notifyConnecting() {

    try {
        if (mWait != null) {
            // 叫醒等待连接的线程
            synchronized (mWait) {
                mWait.notify();
            }
        }
    } catch (Exception e) {
        Utils.printException(e);
    }
}

说明：

  • 服务器连接成功或失败均需叫醒等待线程

@Override
public void close() {

4.3 文件下载

非Http方式时，服务器连接失败（包含重连）,则需要转为Http方式继续下载。

/**
 * 请求资源内容
 * 
 * @param download
 * @param model
 * @return
 */
protected Object[] download(final Download download, final Model model) {

    byte[] bytes = getBytes(download, model);

    Download newDownload = null;
    // 文件下载失败 && 重连服务器失败 && 非Http连接  需要转为 Http下载
    if (bytes == null && download.isNotConnectToServer()
            && !(download instanceof HTTPDownload)) {
        newDownload = Download.createHttp(this, mHost);
        bytes = getBytes(newDownload, model);
    }

    return new Object[] { bytes, newDownload };
}

private byte[] getBytes(Download download, Model model) {
    byte[] bytes = null;
    // 无法连接服务器
    if (download.isNotConnectToServer()) {
        Utils.printLog(Constants.OFF_TAG, "无法连接服务器 " + model.getPath() + "/" + model.getName());
        return bytes;
    }
    try {
        // 下载
        bytes = download.execute(model);

        // websocket 为异步下载
        if (download instanceof WebSocketDownload) {
            WebSocketDownload wsDownload = (WebSocketDownload)download;

            // 再判断是否文件下载中
            synchronized (model) {
                if (wsDownload.isDownloading()) {
                    model.wait();
                } 
            }

            bytes = wsDownload.getBytes();
        } 

        if (bytes != null) {
            if (bytes.length != 1) {// 文件内容
                // 校验
                String shaFile = HMacAdapter.getSHA1(bytes);
                if (Utils.compare(shaFile, model.getRev())) {
                    return bytes;
                } else if (model.isRepDownload()) {
                    // 校验失败重复下载
                    return getBytes(download, model);
                } else {
                    // 校验超过两次时返回null；
                    Utils.printLog(Constants.OFF_TAG,
                            "下载失败(哈希值校验): " + model.getPath() + "/" + model.getName());
                }
            } else {// 错误码
                // 服务器返回失败信息
                ((TCPDownload) download).handleSocketError(Constants.OFF_TAG,
                        bytes[0] & 0x000000ff);
            }
        } else {// 获取失败 网络等异常问题

            if (model.isRepDownload()) {
                // 重复下载
                return getBytes(download, model);
            } else {
                Utils.printLog(Constants.OFF_TAG,
                        "下载失败(网络异常): " + model.getPath() + "/" + model.getName());
            }
        }
    } catch (Exception e) {
        Utils.printLog(Constants.OFF_TAG,
                "下载失败(Exception): " + model.getPath() + "/" + model.getName());
    }
    return null;
}

说明：

  • 由于websocket服务器返回请求内容是异步的，所以此处需要线程同步处理。 如果文件下载中则当前线程等待
