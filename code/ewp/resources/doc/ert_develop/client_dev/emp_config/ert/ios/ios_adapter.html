1 适配方法

需要项目中适配的工作

RYTConfig 类中增加方法+ (CGFloat)fitSize. 内容如下.

    + (CGFloat)fitSize;

    RYTConfig.m

    + (CGFloat)fitSize{

        if ([[RYTConfig sharedRYTConfig] iPhone]) {

            return [[RYTConfig sharedRYTConfig] screenSize].width/320;

                }else{

            return [[RYTConfig sharedRYTConfig] screenSize].width/1024;

        }

    }

上面为RYTConfig中实现的适配方法.通过[RYTConfig fitSize]调用.返回适配参数. 更新RYTConfig类中的- (CGSize)screenSize和- (CGRect)applicationFrame方法

    (CGSize)screenSize { 

    ios 7.1 横屏 width:768 height:1024

    UIDevice *device = [UIDevice currentDevice]; UIInterfaceOrientation orientation = [UIApplication sharedApplication].statusBarOrientation;

    CGRect boudle = [[UIScreen mainScreen] bounds]; CGSize screenSize = boudle.size;

    if ([device.systemVersion floatValue] < 8.0f && (orientation == UIDeviceOrientationLandscapeLeft
    orientation == UIDeviceOrientationLandscapeRight)) {

        screenSize = CGSizeMake(boudle.size.height, boudle.size.width);

    }

    return screenSize;

}

    (CGRect)applicationFrame { 

    UIDevice *device = [UIDevice currentDevice]; CGSize screenSize = [self screenSize]; CGRect applicationFrame = CGRectZero; applicationFrame.size = screenSize; if ([device.systemVersion floatValue] >= 7.0) {

        applicationFrame.origin.y = [RYTConfig statusBarHeight];

    } applicationFrame.size.height = screenSize.height - [RYTConfig statusBarHeight]; return applicationFrame;

}

实现方法中:

  1. screenSize为返回屏幕大小的方法.方法中针对iOS8之前的系统,横屏width和height相反的情况做了处理.

  2. 根据是否是iPhone平台做了判断,如果是/320,否/1024.当然这仅仅是针对iPhone竖屏及iPad横屏做了判断,具体需求,项目上可自行修改参数.

2 适配实现

目前,适配方案具体应用在如下几个方面

  1. 控件大小.即width,height

  2. 控件位置.x,y坐标.bottom,height,left,right属性.

  3. 字体大小,图片大小.

  4. 容器控件的边距和间距

  5. 动画接口中涉及到frame的部分.

  6. 代码中定义的一些默认的宽高等属性.

3 注意事项

  1. 项目上写报文时,不需要考虑机型问题.如报文中,给了一个控件200宽度,以`iPhone竖屏为例,iPhone6之前的机型就是显示200,iPhone6之后的机型根据适配进行相应放大.不用考虑是否iPhone6及之后机型屏幕大了,200是否显小,不合适的情况.该传什么值就传什么值.

  2. 同上例,一个样式wdith为200的控件,我们通过lua方法get控件的宽度时,得到的都会是200.不会因为做了适配,iPhone6及之后机型会返回不一样的值.
