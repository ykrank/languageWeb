1 简介

android所有的可配置项都在EMPConfig.java 类中处理。该类是单例模式，通过EMPConfig.newInstance()可以获取到其唯一的对象实例。

2.2 像素比例配置

字段及默认值

/** 服务器配置的屏幕宽高：该值是屏幕默认基数，需要与服务器端协商一致 */
private static Rect mBasicRect = new Rect(0, 0, 320, 480);
/**资源图片最优显示分辨率宽,默认为480*/
public static  int mImgOptimalBasicWidth = 480;
/** 实际比例=Screen.mWidth/ mBasicRect.width() */
public static float mRealRatio = 1f;

注意：mRealRatio字段为public static类型,该值通过mBasicRect.width与屏幕像素width计算得出

相关方法

  • public Rect getBasicScreen() 获取屏幕默认基数
  • public void setBasicScreen(String) 设置屏幕默认基数并重新计算mRealRatio
  • public void computeRatio() 计算mRealRatio

代码片段

    /**
     * 计算mRealRatio
     */
    public void computeRatio() {
        mRealRatio = Float.intBitsToFloat(Screen.mWidth) / Float.intBitsToFloat(mBasicRect.width());
        Utils.printLog("computeRatio", "----mRealRatio =" + mRealRatio + "   Screen.mWidth=" + Screen.mWidth + "  mBasicRect.width()=" + mBasicRect.width() + "----");
    }

    /**
     * 设置基础屏幕宽高<br>
     * 该值是屏幕默认基数，需要与服务器端协商一致
     * 
     * @param rect
     */
    public void setBasicScreen(Rect rect) {
        if (mBasicRect.equals(rect)) {
            return;
        }
        mBasicRect.set(rect);
        computeRatio();
    }
