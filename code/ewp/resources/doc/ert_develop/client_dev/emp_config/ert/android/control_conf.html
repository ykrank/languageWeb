Android控件配置简介

1 简介

5.2版本中android产品控件的默认样式是通过WidgetConfig.java 以及EmpConfig.java配置的。5.3我们重构了这部分代码，所有配置项均移到了ControlConfig.java中,其他两个类的相关方法已经被废弃，强烈不建议继续使用。

我们在ControlConfig.java中为每个控件都定义了一个相同类名的内部类，在内部类中定义了可配置的公共静态变量。

2 配置方式

在项目的MainActivity中覆写onAdjustWidgetConfig。如配置button点击的delay属性的默认值，代码如下

    @Override
    public void onAdjustWidgetConfig() {
        super.onAdjustWidgetConfig();
        ControlConfig.InputButton.DEF_DELAY=100;
    }

3 坐标相关配置说明

所有与坐标相关的配置变量均与模板样式基准坐标一致，改基准坐标可以通过EmpConfig中配置，默认为320*480。

4 继承关系说明

有继承关系的控件，如果子控件没有某一个默认配置项，但是其父控件有，则父控件的配置会作用到子控件上。如InputText的DEF_WIDTH对InputPassword有作用。
