创建一个WP Application

概述

学习过Client Application开发基础之后，我们就可以试着创建一个最简单的HelloWorld应用程序了。

更改应用程序名称等相关信息

由于本示例仅涉及到加载本地报文，所以就不需要进行配置服务器地址。

打开主项目下Properties目录，双击WMAppManifest.xml从弹出的可视化界面中进行编辑。可视化界面如下图


图1 项目信息修改截图


图2 项目信息修改截图

1 显示名称

显示名称是应用程序在手机中应用程序列表界面显示的名称，将该项目修改为"HelloWorld"。

2 应用程序图标

应用程序图标是应用程序在手机中应用程序列表界面所显示的图片，将该项目修改为需要的图标，该操作之前导入或者替换掉相应的图片文件，该文件默认文件名称为"ApplicationIcon"，在启动项的根目录下。

3 磁贴标题

磁贴标题表示应用程序pin到开始页面上显示的文字，将该项修改为"HelloWorld"。

4 图块图像


图3 桌面磁贴显示模板

图块图像表示应用程序Pin到开始页面上显示的图片，有3种磁贴模版选择：

1）TemplateFlip

磁贴展示，会循环展示图片和文字，所需图片分辨率为 159*159，336*336，691*336（可选）。

2）TemplateIconic

图标模板，需要制定图标大小为710*110，130*202。

3）TemplateCycle

图片循环展示，小图标159*159和至少需要一张分辨率为336*336或者691*336的图片，最多可设置9张图片循环展示。

创建HelloWorld应用程序

用本程序框架创建一个HelloWorld应用将会是一件很简单的事情，

以下分本地报文和服务器报文2种方式：

1 创建基于本地报文的HelloWorld应用程序

1.1 更改应用启动时加载的xml文件

HostPage.xaml.cs文件中，找到AppStart方法，将NavigateWithLocalXml("main.xml")中的main.xml改为Test.xml。

1.2 编辑本地文件Test.xml

从Resources目录下找到Test.xml文件，打开编辑，编写如下报文：

<?xml version="1.0" encoding="UTF-8" ?>
<content>
<head>
<style>
.body1 {background-color:#99000000;}
.lable1 {left:60px;top:200px;font-size:24px;color:#FF0008;font-weight:bold;}
</style>
<script type="text/x-lua" src="RYTL.lua"></script>
</head>
<body class='body1' name='body'>
<label class='lable1' value='Hello World!'>
Hello World!
</label>
</body>
</content>

2 创建基于服务器报文的HelloWorld应用程序

2.1 更改应用启动时加载的xml文件

HostPage.xaml.cs文件中，找到AppStart方法，NavigateWithLocalXml("*****")，将参数改为"main.xml"。

2.2 更改服务器连接地址

打开主项目根目录下的静态类ConfigManager，修改静态变量SERVER_URL_WITH_PORT的值，将其修改为所需要连接的服务器地址。

3 启动调试查看显示效果
