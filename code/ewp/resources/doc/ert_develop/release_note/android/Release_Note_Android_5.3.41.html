1.1 离线资源2.0

  1. 支持在服务端配置离线资源安全传输的方式：明文传输、EMP信道明文传输、EMP信道密文传输、HTTPS/SSL；
  2. 服务端根据配置在描述文件中增加传输方式字段，客户端根据该字段标识的传输方式下载资源。不符合规定的下载请求将被拒绝；
  3. 修改现有HTTP和TCP下载接口，使其支持四种传输方式；
  4. 修改服务端存储离线资源的路径，使资源不会被直接下载；
  5. 兼容旧版本客户端；
  6. 支持下载进度。

2 api修改

2.1 公共功能变更

2.1.1 重构键盘部分代码

参见Android 键盘开发指导

由于键盘的变更影响到的控件有inputtext、inputpassword、select。(之前多选框键盘代码融合在select中，现在抽出为SelectKeyBoardHelper.java)

2.1.2 Property公共类

一些属性为多个控件公用，在每个控件中均处理一次增加了冗余代码，因此抽出公共类来处理这样的属性。这些类均放在包com.rytong.gui.atom.property下。

2.1.2.1 新增类说明
类名 功能
PropertyTextEffects 处理文字特殊效果（阴影、跑马灯、省略号等）
PropertyImg 处理Button和InputText的leftImg和rightImg属性
PropertyDelay 处理Button的delay属性
PropertyBorder 处理支持border属性控件的border

2.1.3 SetBg接口变更

之前的SetBg接口仅可以处理Drawable,为了可以处理二进制文件，并且兼容之前的SetBg，新增接口HandleRes。之前使用SetBg接口的文件不需要做改变。

public interface HandleRes {

    /**
     * <p>
     * 加载二进制资源后成功后处理资源
     * </p>
     * 
     * @param isSyn 背景图片是否同步方式获取
     * @param byteData 二进制数据
     */
    public void handleResOnSuccess(boolean isSyn, byte[] byteData);

    /**
     * <p>
     * 处理加载失败的情况
     * </p>
     * 
     * @param isSyn 背景图片是否同步方式获取
     * @param byteData 二进制数据
     */
    public void handleResOnFail(boolean isSyn);

}

3 交付在升级时需注意的问题

3.1 国际化

使用国际化功能时，需要将项目自定义的string.xml文件做国际化处理

3.2 slt2脚本解析功能

为使用slt2脚本解析功能，需要在assets目录下放置slt2的解析lua文件SLT2content.lua

3.3 高德地图升级

升级后的高德地图更换了release key，需要项目上自己去高德官网申请（或者使用产品申请的key），否则会报INVALID_USER_SCODE提示，同时无法使用地图功能
