1 CSS说明

ERT针对每一个HTML标签，从标准CSS规范中选取了针对该标签适合在移动终端使用的CSS样式子集。通过ERT CSS引擎让这些CSS样式可以应用到其所支持的HTML标签上。这样，界面上的各种控件就能展示出不同的样式和效果。

2 标准CSS比对

ERT CSS支持标准CSS的基础语法，但是选择器前必须有.并且颜色值只支持16进制。

对于标准CSS的高级语法，目前ERT CSS不支持的有：

  • 选择器分组
  • 派生选择器
  • id 选择器
  • 属性选择器

支持的有：

  • 继承
  • 类选择器

3 定位

ERT CSS使用的是绝对定位，元素框相对于其包含块定位。对于未写绝对位置的元素，ERT CSS会和标准CSS普通流定位一样自动排版。

4 支持的CSS样式

客户端所支持的CSS样式及详细介绍请参照EMP参考手册CSS样式介绍。

5 Examples

这里我们给出三个简单示例以帮助理解ERT CSS的用法:

  • 单一样式: 每个控件引用一个样式，例: <label class="style1">example</label>。
  • 多重样式: 每个控件可引用多个样式，例: <label class="style1,style2">example</label>。
  • 引用样式: 通过引用外部的样式文件获取样式。

5.1 单一样式

示例代码：

<content>
<head>
    <style>
        .body {background-color: #FFFFFF;}
        .div1 {background-color: #8AB813;left:60px; top:30px; height:300px; width:200px; font-size:17px;}
        .label {color: #FF00FF;}
        .label1 {top:10px; left:20px; font-weight: bold; font-size: 11px; width: 160px; text-align: center;}
        .input1 {left:20px; width:160px; height:40px;}
        .button1{left:20px; width:160px; background-color: #F5DE01;}
    </style>
</head>
<body>
    <div class="div1">
        <label class="label1">CSS简单例子</label><br/>
        <input type="text" class="input1"></input><br/>
        <input type="button" class="button1" value="样式测试"></input>
    </div>
</body>
</content>

运行效果图：

说明:

  • 代码中的.label属于元素选择器，元素选择器定义的样式适用于所有具有相同标签名的元素。
  • 类选择器的优先级大于元素选择器，故若出现两种选择器同时作用于同一标签时，类选择器的属性覆盖元素选择器的相同的属性。
  • 样式具有继承性: 如果父节点的样式中定义了子节点样式没有定义的CSS属性，那么该CSS属性对子节点起作用。可以继承的样式有: font-size,font-weight,color,font-style。

根据运行效果图可以看出:

  • label控件使用了元素选择器.label和类选择器.label1的合集样式: 字体color样式来自于.label，其他来自于.label1；
  • button控件的font-size样式继承于其父控件div的样式.div1；label控件则使用了自己的font-size样式。

5.2 多重样式

示例代码：

<content>
<head>
    <style>
        .body {background-color: #FFFFFF;}
        .div1 {background-color: #8AB813;left:60px; top:30px; height:300px; width:200px; font-size:17px;}
        .label {color: #FF00FF;}
        .label1 {top:10px; left:20px; font-weight: bold; font-size: 11px; width: 160px; text-align: center;}
        .input1 {left:20px; width:160px; height:40px;}
        .button1{left:20px; width:160px; background-color: #F5DE01;}
        .button2{height:40px; width:100px; top:100px;}
    </style>
</head>
<body>
    <div class="div1">
        <label class="label1">CSS简单例子</label><br/>
        <input type="text" class="input1"></input><br/>
        <input type="button" class="button1,button2" value="样式测试"></input>
    </div>
</body>
</content>

运行效果图：

说明:
当控件引用了多个样式时，控件的最终样式为多个样式的合集，若合并过程中存在相同的属性，则被最后引用的该属性样式生效。

从运行效果图中可以看到：
button控件使用了.button1和.button2定义的样式合集。对于这两个样式中重复的样式以后面一个样式生效。此例中为width。

5.3 引用样式

示例代码：

<content>
<head>
    <link ref="css_test.css" type="text/css"/>
    <style>
    </style>
</head>
<body>
    <div class="div1">
        <label class="label1">CSS简单例子</label><br/>
        <input type="text" class="input1"></input><br/>
        <input type="button" class="button1" value="样式测试"></input>
    </div>
</body>
</content>

css_test.css代码：

.body {background-color: #FFFFFF;}
.input1 {left:20px; width:160px; height:40px;}
.label {color: #FF00FF;}
.label1 {top:10px; left:20px; font-weight: bold; font-size: 11px; width: 160px; text-align: center;}
.button1{left:20px; width:160px; background-color: #F5DE01;}
.div1 {background-color: #8AB813;left:60px; top:30px; height:300px; width:200px; font-size:17px;}

运行效果图：

此例中将style放入了css_test.css文件中，并通过link标签引入样式表，实现了与示例一相同的效果。
