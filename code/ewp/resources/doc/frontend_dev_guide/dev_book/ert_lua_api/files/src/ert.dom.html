Functions

dom:get_ctrl (...)
根据属性列表获取控件，如果控件存在则返回控件列表，如果控件不存在则弹出提示框，返回nil。

Parameters

  • ...: 属性列表，例如{name="button",type="submit"}

Return value:

table 控件列表
dom:get_ctrl_by_id (id)
根据id获取控件，如果控件存在则返回根据id获取的控件，如果id重复也只返回获取的第一个控件，如果控件不存在则弹出提示框，返回nil。

Parameters

  • id: string 控件id

Return value:

userdata 单个控件

dom:set_style_by_id (id, style, value)
根据id改变控件样式，只修改界面中有此id的第一个控件。

Parameters

  • id: 控件Id
  • style: 样式名称，比如："height"
  • value: 修改值，需要将控件样式修改的值

Return value:

nil
dom:set_style_by_name (name, style, value)
根据名称改变控件样式，如果界面中有多个此名称控件则这些控件都会被修改。

Parameters

  • name: 控件名称
  • style: 样式名称，比如："height"
  • value: 修改值，需要将控件样式修改的值

Return value:

nil
