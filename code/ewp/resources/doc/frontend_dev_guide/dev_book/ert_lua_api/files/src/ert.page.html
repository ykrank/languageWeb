Tables

Page
Page类，每个实例对应Channel的一个界面，保存在所属Channel的pages字段中。Page实例一般会 被每个界面的lua 脚本引用，通常为this = ert.channel:get_page(id, trancode);

Fields

  • _id: 当前流程所对应的channel id
  • _trancode: 当前流程所处的channel 步骤代码
  • _content: Page报文，通常为静态html界面
  • _data: Page输入 = ert.channel:get_response(_id, _trancode);
  • _user_input: 以kv形式存储Page页面中用户输入的信息，key为页面标签的id
  • _last_page: 同一Channel流程中的上一个Page实例，用于Page:replace()时缓存其 页面上的用户输入
