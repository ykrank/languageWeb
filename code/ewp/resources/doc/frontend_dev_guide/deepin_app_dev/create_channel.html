创建channel

1 概述

本章主要介绍channel的创建过程。

2 创建channel流程

在EMP App 中选择Show Channel，在channel management 栏位下点击add按钮添加channel。 填写自己的Channel Id 及 Channel Name，其他为默认状态。例：如下图所示添加一个my_account的channel。

点击ok正常添加成功后在resouce_dev/common/channels文件夹下便可看到一个名为my_account文件夹。

my_account文件夹下有对应生成的css，images，json，lua，xhtml文件夹。

点击Show Channel，选择对应的Channel Name此处为“我的账户”后点击Edit 进入编辑页面。

在编辑页面，点击按钮“Add a step”逐个添加页面。

添加我的账户所需三个步骤，暂定为MB0001，MB0002，MB0003，具体步骤ID为前端开发人员和后台开发 人员商议确定。在TranCode、Adaper、Proceduer输入框内分别填入MB0001/MB0002/MB0003、simulator、 MB0001/MB0002/MB0003，在Viewr输入框内会自动生成my_account_MB0001/MB0002/MB0003,点击OK添加完成。

完成后，可以看到json,lua,xhtml文件夹下都自动生成了对应步骤ID的文件。

启动模拟器，运行项目。在项目入口界面处的Channel Id输入框内填写“my_account”， 在TranCode输入框内填写“MB0001”，如下图所示：

点击“访问”按钮 进入我的账户第一个界面，即MB0001.xhtml。
